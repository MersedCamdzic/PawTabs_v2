================================================================
PawTabs v2.1.0 — Screenshot recipe for Chrome Web Store
================================================================

Setup (fully automated now — no manual tab opening needed)
----------------------------------------------------------
1. Load unpacked dist/ into Chrome (chrome://extensions →
   Developer mode → Load unpacked → select the dist/ folder).
2. Open Mission Control (click PawTabs icon in Chrome, then the
   GridFour icon at the top-right of the popup — or navigate to
   chrome-extension://<id>/src/mission-control/index.html).
3. Scroll to the bottom of the Overview page → "Demo / testing"
   card.
4. Click "Load demo data" — populates paw / tags / notes /
   snapshots / backups / window colors. Confirms with a dialog.
5. Click "Enter demo mode" — saves your current session as
   "Pre-demo backup", closes all your open windows, then opens
   6 named+colored demo windows (~90 tabs total):
     - Posao (blue) — Gmail, Calendar, GitHub, Linear, Notion,
       Figma, Vercel, Slack, Jira, ClickUp, Trello, Meet, Drive
     - Research (green) — arXiv, Wikipedia, MDN, StackOverflow,
       Hacker News, Medium, ChatGPT, Claude, Perplexity
     - Vikend (amber) — Amazon, CNN, BBC, NYT, YouTube, Netflix,
       Reddit, Instagram, X, LinkedIn, Pinterest, Airbnb
     - Learning (purple) — Coursera, Udemy, freeCodeCamp,
       egghead
     - Finance (red) — Chase, Fidelity, WSJ, Bloomberg, CoinGecko
     - Music (pink) — Spotify, YouTube Music, SoundCloud,
       Bandcamp, Apple Music
6. Set Chrome zoom to 100% (Cmd+0) and resize the browser to
   ~1440×900 so the popup renders cleanly at 480px.

Web Store screenshot spec: 1280×800 PNG, max 5.
Recommended macOS tool: Cmd+Shift+4 → Space → click window
(gives a clean shot with drop shadow).

5 SCREENSHOTS TO CAPTURE (in this order)
================================================================

--- Screenshot 1: Popup with grouped-by-window view ---
Purpose: shows the everyday interface + window color theming.
Steps:
  1. Focus the Vikend (amber) demo window so it becomes "current".
  2. Click PawTabs icon.
  3. Grouping: Window. Search: empty.
  4. Hover over the "POSAO (13)" group header so the group
     actions (Tag / Note / Move / Save-snapshot / X) fade in.
  5. Capture the popup only (Cmd+Shift+4 → Space → click popup).
Composition: colored group headers stacked (Posao blue / Vikend
  amber / Research green / etc), inline actions visible, current
  tab card at top with pulsing green dot.

--- Screenshot 2: Mission Control Overview ---
Purpose: shows the "cockpit" and richness of data.
Steps:
  1. Open Pack (Mission Control) from popup grid icon.
  2. Sidebar visible on left, Overview view centered.
  3. Screenshot the whole Chrome window.
Composition: sidebar with counts + Overview cards — Windows /
  Open (~90) / Inactive stats at top, Collections tiles (Pinned,
  Pawed, Audible, Tagged), Top domains chart, timeline.

--- Screenshot 3: Mission Control → Windows view ---
Purpose: shows window naming + color theming + per-window tabs.
Steps:
  1. Sidebar → Collections → Windows
  2. Columns: 2 or 3 (top-right of toolbar)
  3. All 6 demo windows are already named+colored.
  4. Capture the Chrome window.
Composition: named window cards with subtle colored borders,
  header ribbons in each color, compact tab rows with state
  icons + tag/note counts inside.

--- Screenshot 4: Tab details modal ---
Purpose: shows the depth per tab (state, paw, pin, tags, notes,
  window, move-to-window).
Steps:
  1. In popup, click any tab row (NOT the ListBullets info
     button — that opens the same modal but with clicked-row
     targeting).
  2. Modal opens with subtitle showing window name.
  3. In the "Move to window" section, click the dropdown so the
     colored window entries are visible.
  4. Capture the whole popup.
Composition: header actions row (Paw / Pin / Mute? / Jump /
  Close-X), URL bar with copy, Tags section (purple), Notes
  (cyan card), Move-to-window dropdown with color dots.

--- Screenshot 5: Tags view ---
Purpose: shows persistent lists (Reading / Research / Work /
  Learning / Watch later / Music / Social / Finance / Travel).
Steps:
  1. Mission Control → Collections → Tags
  2. Click the "Work" tag in the sidebar (fullest with demo
     data — has 20+ URLs including all the Notion/GitHub/Figma
     tabs).
  3. Right panel shows tagged URLs with state icons + paw/pin +
     window chip (colored).
  4. Capture.
Composition: sidebar with tag list (each with count badge) +
  right panel with tagged URLs, each row showing state icon,
  paw/pin/tag/note icons, and window chip in matching color.

================================================================
POST-SHOOT
================================================================

After screenshots done:
  [ ] Mission Control → Overview → Demo card →
      "Exit demo & restore session" — closes demo windows and
      restores your original tabs from the Pre-demo backup.
  [ ] (Optional) "Clear all data" in the same Demo card resets
      storage to a fresh state.
  [ ] Rename PNGs (recommended):
      01-popup.png, 02-overview.png, 03-windows.png,
      04-details.png, 05-tags.png
  [ ] Verify each PNG is exactly 1280×800 (crop or downscale if
      Chrome captured retina 2x resolution — most Cmd+Shift+4
      shots are 2x on Retina; use Preview.app → Tools → Adjust
      Size → 1280×800 → Save).
  [ ] Upload in order in Chrome Web Store dashboard →
      Store listing → Screenshots.

================================================================
OPTIONAL 6TH: Recently Closed with rich data
================================================================

If you want a bonus shot:
  1. Close ~5 tabs from your demo windows (Cmd+W on tabs like
     Amazon, GitHub PR, CNN article).
  2. Mission Control → Browser history → Recently closed
  3. Each closed URL now shows Prohibit red state icon +
     PawPrint (if pawed) + tag count (purple) + note count
     (cyan) — the closed-URL detail modal is fully featured too.
  4. Capture.

Chrome Web Store accepts up to 5 screenshots so pick your best 5.
